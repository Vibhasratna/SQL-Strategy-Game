SchemaVerse: SQL Strategy Game

🚀 How to Play SchemaVerse

Welcome to SchemaVerse, a space-based SQL strategy game where you command fleets using SQL commands! Your goal is to dominate the galaxy by strategically moving ships, attacking enemies, and mining resources.

📜 Gameplay Instructions:

  1. Join the Game: Sign up or log in to start playing.
  2. View Your Fleet: Your ships are displayed on the game board.
  3. Command Your Ships: Use SQL queries in the command console to control your fleet.
  4. Attack Enemies: Reduce enemy ship HP using SQL update commands.
  5. Mine Resources: Gather resources strategically to strengthen your fleet.
  6. Win the Game: The last player standing wins the round!

💻 SQL Commands:

  • SELECT * FROM ships WHERE player = 'YourName'; ➝ View your ships
  • UPDATE ships SET position_x = 5, position_y = 7 WHERE id = 3; ➝ Move a ship
  • UPDATE ships SET strength = strength - 10 WHERE id = 4; ➝ Attack an enemy
  • DELETE FROM ships WHERE strength <= 0; ➝ Destroy weak ships

🏆 Winning Conditions:

Eliminate all enemy ships to be the last player standing. The winner earns bonus points on the leaderboard!

🎮 Tips & Strategies:

  • Move strategically to avoid enemy attacks.
  • Monitor your ship’s strength and retreat if necessary.
  • Use SQL efficiently to issue quick commands.
